# Cybersecurity Labs - Danys Díaz Ruiz

Repositorio de prácticas y laboratorios de ciberseguridad enfocados en el uso de herramientas, análisis básico de vulnerabilidades y comprensión de técnicas comunes en entornos controlados.

Este portafolio forma parte de mi proceso de formación como Técnico en Ciberseguridad en CENFOTEC.

## Objetivo

Documentar y demostrar, de forma práctica, mis conocimientos en:
- Fundamentos de ciberseguridad.
- Uso de herramientas de análisis y pruebas.
- Metodología básica de laboratorio.
- Buenas prácticas y aprendizaje continuo.

Todos los ejercicios se realizan en **entornos de laboratorio controlados** con fines educativos.

---

## Herramientas utilizadas

- Kali Linux  
- Metasploitable 2  
- Nmap / Zenmap  
- Hydra  
- Wireshark  
- Metasploit Framework  
- VirtualBox  

---

## Laboratorios incluidos

Este repositorio contendrá, entre otros:

- Escaneo de puertos y servicios con Nmap.  
- Ataques de fuerza bruta con Hydra en entornos vulnerables.  
- Análisis de tráfico con Wireshark.  
- Pruebas básicas con Metasploit.  
- Simulación de ataques Man-in-the-Middle (MITM).  

Cada laboratorio incluye:
- Descripción del objetivo.  
- Entorno utilizado.  
- Pasos realizados.  
- Evidencia (capturas).  
- Aprendizajes.

---

## Aviso

Todo el contenido de este repositorio se realiza con fines **educativos** en laboratorios locales.  
No se promueve ni se apoya el uso de estas técnicas en sistemas sin autorización.
